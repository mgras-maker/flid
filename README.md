# FLID Foundation Website

A beautiful, creative, and accessible interactive application for the FLID foundation (Foundation People-Innovations-Design), focused on balance through design and designing positive change for sustainability.

## Features

- Modern, responsive design with subtle animations
- Beautiful styling that matches the FLID color scheme (#D2CDE8)
- Interactive 3D visualization of the design thinking process
- Project showcase with animated cards
- Fully accessible and mobile-friendly

## Technologies Used

- **Frontend Framework**: React with TypeScript
- **Build Tool**: Vite for fast development and optimized builds
- **Styling**: Tailwind CSS for responsive design
- **Animations**: Framer Motion for smooth transitions and interactions
- **3D Elements**: React Three Fiber and Drei for 3D visualizations
- **Routing**: React Router for navigation between pages

## Deployment

This website is deployed on GitHub Pages at [https://mgras-maker.github.io/flid/](https://mgras-maker.github.io/flid/)

### Manual Deployment

To deploy the website to GitHub Pages:

1. Clone the repository:
   ```
   git clone https://github.com/mgras-maker/flid.git
   cd flid
   ```

2. Install dependencies:
   ```
   npm install
   ```

3. Run the deployment script:
   ```
   npm run deploy
   ```

### Automated Deployment

The site is automatically deployed to GitHub Pages when changes are pushed to the main branch, using GitHub Actions.

## Getting Started

### Prerequisites

- Node.js (v14 or newer)
- npm or yarn package manager

### Installation

1. Clone the repository
   ```
   git clone <repository-url>
   cd flid-website
   ```

2. Install dependencies
   ```
   npm install
   ```

3. Start the development server
   ```
   npm run dev
   ```

4. Build for production
   ```
   npm run build
   ```
